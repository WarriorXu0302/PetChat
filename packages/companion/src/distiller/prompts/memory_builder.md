# 共同记忆生成模板 (通用联系人版)

把 `memory_analyzer.md` 的提取结果组织成可直接嵌入 `SKILL.md` Part A 的 markdown。

## 模板

```markdown
## 关系时间线
- 认识: {when_where}
- 关系变化: {milestones}
- 当前状态: {status_one_liner}

## 共同场景
{key_projects_or_contexts}

## 交流节奏
- 主动性: {who_initiates_more}
- 活跃时段: {active_hours_summary}
- 话题分布: {topic_mix}

## Inside Jokes / 默契
{repeatable_memes_and_callbacks}

## 工作 / 技术上下文
{shared_tools_and_jargon}

## 分歧与合作模式
{observed_conflict_patterns}

## 高光时刻
{shared_achievements_or_help}

## Correction 记录
(用户纠正后追加的内容, 初始为空)
```

## 写作要求

1. 每节必须有内容; 没信息的节写 `[信息不足]` 一行即可, 不要编。
2. 具体时间、项目名、术语不要泛化 —— "那个大改版" 要换成真名。
3. Inside Jokes 用原文复述, 不要解释它为什么好笑。
4. 涉及第三方的隐私信息只保留存在痕迹 (例: "ta 提过一次家里的事, 没展开"), 不要复制具体内容。
