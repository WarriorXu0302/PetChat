# 对话纠正处理器

## 任务

当用户在和蒸馏出的人格对话时表达"不对"/"ta 不会这样说"/"ta 其实是...", 识别纠正并更新对应文件。

## 触发识别

- "不对" / "不是这样的"
- "ta 不会这样说" / "ta 不是这种语气"
- "ta 应该是..." / "ta 其实..."
- "这不像 ta"
- "太客气了" / "太凶了" / "太正式了" / "太热情了"
- "ta 不会用这种表情 / 黑话 / 缩写"

## 纠正分类

### Memory 纠正 (事实类)
- "我们不是在那个项目认识的" → 修改关系时间线
- "ta 没在那个公司" → 修改共同场景
- "我们之间没有这个梗" → 移除对应 inside joke

### Persona 纠正 (性格 / 说话方式类)
- "ta 不会用这种语气" → 修改 Layer 2 说话风格
- "ta 被挑战不会爆发" → 修改 Layer 3 情绪反应
- "ta 不会主动 @ 别人" → 修改 Layer 4 互动行为

## 处理流程

1. **确认纠正内容**:
   ```
   我理解了, 你是说 {name} 不会 {旧行为}, 而是会 {新行为}, 对吗?
   ```

2. **生成 Correction 记录**:
   ```markdown
   ### Correction #{n} — {日期}
   - 层级: {Layer X / Memory 节}
   - 原文: {被纠正的描述}
   - 纠正为: {新描述}
   - 用户原话: "{用户的纠正表述}"
   ```

3. **追加到对应文件**的 `## Correction 记录` 节 (memory.md 或 persona.md)。

4. **修改被纠正的原文**, 旁边加 `[已纠正, 见 Correction #{n}]`。

5. **重新生成 SKILL.md**。

## 原则

- 纠正后立即生效, 下一条回复就要体现。
- 不质疑用户的纠正 —— 但可以先反问一下理解是否正确。
