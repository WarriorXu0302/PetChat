# 信息录入引导脚本

## 开场白

```
我会帮你把一个联系人(同事/朋友/家人)蒸馏成一个可以对话的 AI 人格。
过程是 3 个问题 + 一份飞书聊天记录(可选)。
```

## 问题序列

### Q1: 代号 (必填)

```
先给 ta 起个代号 —— 昵称、备注名、花名都行。
slug 会用代号生成: 中文转拼音, 英文小写, 空格换下划线。
示例: 小李 / 产品-王总 / xiaoming
```

验证: 非空。

### Q2: 关系与基本信息 (可跳过)

```
一句话说明: 你和 ta 的关系是? ta 做什么的? 认识多久?

示例:
  "大学同学 现在在字节做前端 认识 8 年"
  "我直属上级 PM 我们 1v1 一年多"
  "小区邻居 退休教师 偶尔聊聊"
```

解析字段:
- `relation`: 关系 (同事/朋友/上下级/家人/其他)
- `occupation`: 职业
- `context`: 共同背景 (同一个公司/项目/群)
- `known_duration`: 认识时长

### Q3: 性格画像 (可跳过)

```
一句话描述 ta 的性格 / 说话方式 / 让你印象最深的点。
MBTI、星座、口头禅、做事风格都行。

示例:
  "INTP 技术思维 讲话爱用英文缩写 懒得废话"
  "E 人 话很多 情绪写在脸上 凡事先表态再想"
  "老派东北大哥 爱讲段子 但真有事很靠谱"
```

解析字段:
- `mbti`, `zodiac`
- `personality_tags`
- `impression`: 主观印象

## 汇总确认

```
整理一下:
  代号: {name}
  关系: {relation_summary}
  性格: {personality_summary}
确认后进入下一步: 导入飞书聊天记录。
```
